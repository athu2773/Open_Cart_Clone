How to install OpenCart

Requirements

These are minimum requirements needed so that OpenCart can be installed and work properly.

  • ✅ Web Server (Apache suggested)
  • ✅ PHP (at least 5.4)
  • ✅ Curl enabled
  • ✅ Database (MySQLi suggested)

1

Download OpenCart

The latest version of OpenCart can be downloaded from the OpenCart website (recommended).

2

Upload files to your server

Upload to your web server and extract the OpenCart archive. Use FTP to transfer files.

3

Run automatic installer

Go to the OpenCart location and it will install automatically.
